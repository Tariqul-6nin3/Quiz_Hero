Q.1: Write down the difference among Map, Filter, ForEach and Find?
Main difference among those methods of javascript:

  • Map: This method creates a new array with the results of calling a function on every element in the array. It returns a new array with the same length as the original array.
  • Filter: This method creates a new array with all the elements that pass the test implemented by the provided function. It returns a new array with only the elements that meet the specified condition.
  • ForEach: This method calls a function for each element in the array. It does not return anything, it simply iterates over the array and performs the specified action on each element.
  • Find: This method returns the value of the first element in the array that satisfies the provided function. It returns undefined if no element satisfies the condition.

    Definition of Event Bubble: What is event bubble? Event bubbling is such a concepts that, when an event receives a command to the innermost target element and that events bubble up (or we can say is transmitted or propagated) to its parants untill the root element come through a DOM tree. The major use of event bubbling is the registration of default functions present in the program.Event bubbling is important in web development because it allows developers to create more modular and reusable code by attaching event listeners to parent elements rather than individual child elements. However, it can also lead to unexpected behavior and performance issues if not managed properly.
